Events

Uz pomož jQuery-ja možemo lako da odgovorimo na događaju piput klika, mouseenter, mouseleave....

  • click()
  • dbclick()
  • mouseenter()
  • mouseleave()
  • mousedown()
  • mouseup()
  • mousemove()
  • hover()
  • focus()
  • blur()
  • keydown()
  • keyup()
  • keypress()

click()

$('#clickButton').on('click', function() {
    alert('CLICK');
});

dblclick()

$('#dblclickButton').on('dblclick', function() {
    alert('DOUBLE CLICK');
});

mouseenter()

$('#mouseenterButton').on('mouseenter', function() {
    alert('MOUSE ENTER');
});

mouseleave()

$('#mouseleaveButton').on('mouseleave', function() {
    alert('MOUSE LEAVE');
});

mousedown()

$('#mousedownButton').on('mousedown', function() {
    alert('MOUSE DOWN');
});

mouseup()

$('#mouseupButton').on('mouseup', function() {
    alert('MOUSE UP');
});

mousemove()

$('#mousemoveButton').on('mousemove', function() {
    alert('MOUSE MOVE');
});

hover()

Prva funkcija se poziva ma mouseenter, a druga na mouseleave 

$('#hoverButton').hover(
    function() {
        alert('MOUSE ENTERED');
    },
    function() {
        alert('MOUSE LEFT');
    }
);

focus()

$('#focusButton').on('click', function(){
    $('#focusInput').focus();
});

blur()

$('#blurInput').on('blur', function(){
    alert('BLURED');
});

keydown()

Poziva se kad se pritisne dugme, 

$('#keydownInput').on('keydown', function(){
    alert('KEY DOWN');
});

keyup()

Poziva se kad se pritisne pa pusti dugme (element mora imati fokus). 

$('#keyupInput').on('keyup', function(){
    alert('KEY UP');
});

keypress()

Poziva se kad se pritisne dugme pa pusti i element ostane u focusu. Probati tab dugme. 

$('#keypressInput').on('keypress', function(){
    alert('KEY PRESS');
});
