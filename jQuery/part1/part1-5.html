text()

Ova funkcija uzima klot text iz selektovanog html elementa. Dakle briše sve html tagove. Ako se text funckiji prosledi string onda se sadrzaj elenta setuje na taj string 

//HTML
<p id="ptag">
    <span>Text</span>
</p>

//js
var text = $('#ptag').text(); // sadrzaj promenljive text je Text

html()

Radi slično kao text() samo što uzima kompletan sadržaj selektovanog elementa sa sve tagovima i njegovim atributima 

//HTML
<p id="ptag">
    <span>Text</span>
</p>

//js
var text = $('#ptag').html(); // sadrzaj promenljive text je <span>Text</span>

val()

val() uzima, dok val(x) setuje vrednost input polja. 

var val = $('#inputField').val(); //uzima vrednost
var val = $('#inputField').val('new vale'); //setuje vrednost

attr()

U zavisnosti od prosleđenih parametara može da uzme ili setuje vrednost attribita selektovanog elementa. 

//uzima vrednost attributa height
var height = $('#theDiv').attr('height');
//setuje vrednost height atributa
$('#theDiv').attr('height', '200'); 
//setuje vrednost više atributa(JavaScript objekat)
$('#theDiv').attr(
    {
     height, 200,
     width, 500,
    }
);

removeAttr()

Uklanja atribut. 

$('#theDiv').removeAttr('height');

prepend()

Insertuje prosleđeni html na početak selektovanog elementa. Postojeći html sadržaj se dodaje na novi. 

$('#theDiv').prepend('<b>Prepend</b>');

append()

Dodaje prosleđeni html na kraj selektovanog elementa. Na postojeći html sadržaj dodaje se novi. 

$('#theDiv').append('<b>Append</b>');

before()

Dodaje prosleđeni html ispred selektovang elementa u odnosu na njegov parent. 

$('#theDiv').before('<b>Before</b>');

after()

Dodaje prosleđeni html iza selektovang elementa u odnosu na njegov parent. 

$('#theDiv').after('<b>After</b>');

insertBefore()

Pomera selektovan element na ispred elementa čiji je selektor prosleđen kao parametar funkcije insertBefore. 

$('#theDiv').insertBefore('#anotherDiv');

insertAfter()

Pomera selektovan element iza elementa čiji je selektor prosleđen kao parametar funkcije insertAfter. 

$('#theDiv').insertAfter('#anotherDiv');

wrap()

"Umota" selectovan element u prosleđen HTML 

$('#theDiv').wrap('<div class="wrap" height="250"></div>');

//rezultat
<div class="wrap" height="250">
    <div id="theDiv">
        .....
    </div>
</div>

replaceWith()

Zameni selectovan element sa prosleđenim HTML stringom 

$('#theDiv').replaceWith('<div id="newDiv"></div>');

U ovom primeru div sa id="theDiv" se zamenjuje sa div-om sa id="newDiv". Ovaj poziv će zameni unutrašnji HTML.

replaceAll()

Radi isto što i replaceWith samo sa "naopakom" sintaksom. 

$('<div id="newDiv"></div>').replaceAll('#theDiv');

remove()

Briše selektovan element. 

$('#theDiv').remove();

empty()

Briše sav sadržaj selektovanog elementa. 

$('#theDiv').empty();

clone()

Pravi kopiju selektovanog elementa. 

var clone = $('#theDiv').clone();

Primer odozgo neće iskopirati event hendlere. Da biste i njih iskopirali prosleđuje se true u clone funkciju 

var clone = $('#theDiv').clone(true);

data()

data() funkcija omogućuje da se na DOM element doda JavaScript objekat. 

//dodavanje JS objekta
var theObject = {property : 'value'};
$('#theDiv').data('key', theObject);

//uzimanje objekta
var object = $('#theDiv').data('key');

//brisanje objekta
$('#theDiv').removeData('key');
